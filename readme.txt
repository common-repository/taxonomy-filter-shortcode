A responsive, shortcode taxonomy filter.

== Description ==

A responsive, shortcode taxonomy filter that allows you to filter posts on a page by taxonomy or transfer you to the taxonomy page on a mobile. The settings allow you to decide whether to hide or show the all and no count options, as well as setting up a paging system to load more posts on scroll.

== Installation ==

1. Upload `taxonomy-filter-shortcode` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Are there any settings available to me? =

Yes. You'll find some options under the Settings->Taxonomy Filter page.
